# Clari Skill Pack

> Claude Code skill pack for Clari integration (18 skills)

## Installation

```bash
/plugin install clari-pack@claude-code-plugins-plus
```

## Usage

Skills trigger automatically when you discuss Clari topics. For example:

- "Help me set up Clari" → triggers `clari-install-auth`
- "Debug this Clari error" → triggers `clari-common-errors`
- "Deploy my Clari integration" → triggers `clari-deploy-integration`
