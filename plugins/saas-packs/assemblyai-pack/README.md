# AssemblyAI Skill Pack

> Claude Code skill pack for AssemblyAI integration (18 skills)

## Installation

```bash
/plugin install assemblyai-pack@claude-code-plugins-plus
```

## Usage

Skills trigger automatically when you discuss AssemblyAI topics. For example:

- "Help me set up AssemblyAI" → triggers `assemblyai-install-auth`
- "Debug this AssemblyAI error" → triggers `assemblyai-common-errors`
- "Deploy my AssemblyAI integration" → triggers `assemblyai-deploy-integration`
