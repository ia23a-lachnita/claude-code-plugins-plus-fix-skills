# StackBlitz Skill Pack

> Claude Code skill pack for StackBlitz integration (18 skills)

## Installation

```bash
/plugin install stackblitz-pack@claude-code-plugins-plus
```

## Usage

Skills trigger automatically when you discuss StackBlitz topics. For example:

- "Help me set up StackBlitz" → triggers `stackblitz-install-auth`
- "Debug this StackBlitz error" → triggers `stackblitz-common-errors`
- "Deploy my StackBlitz integration" → triggers `stackblitz-deploy-integration`
