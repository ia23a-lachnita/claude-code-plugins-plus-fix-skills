## Installation

```bash
/plugin install figma-pack@claude-code-plugins-plus
```

## Usage

Skills trigger automatically when you discuss Figma topics. For example:

- "Help me set up Figma" → triggers `figma-install-auth`
- "Debug this Figma error" → triggers `figma-common-errors`
- "Deploy my Figma integration" → triggers `figma-deploy-integration`
