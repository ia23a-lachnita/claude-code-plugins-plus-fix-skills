# Wispr Skill Pack

> Claude Code skill pack for Wispr integration (18 skills)

## Installation

```bash
/plugin install wispr-pack@claude-code-plugins-plus
```

## Usage

Skills trigger automatically when you discuss Wispr topics. For example:

- "Help me set up Wispr" → triggers `wispr-install-auth`
- "Debug this Wispr error" → triggers `wispr-common-errors`
- "Deploy my Wispr integration" → triggers `wispr-deploy-integration`
