# BambooHR Skill Pack

> Claude Code skill pack for BambooHR integration (18 skills)

## Installation

```bash
/plugin install bamboohr-pack@claude-code-plugins-plus
```

## Usage

Skills trigger automatically when you discuss BambooHR topics. For example:

- "Help me set up BambooHR" → triggers `bamboohr-install-auth`
- "Debug this BambooHR error" → triggers `bamboohr-common-errors`
- "Deploy my BambooHR integration" → triggers `bamboohr-deploy-integration`
