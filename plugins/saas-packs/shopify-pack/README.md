```bash
/plugin install shopify-pack@claude-code-plugins-plus
```

## Usage

Skills trigger automatically when you discuss Shopify topics. For example:

- "Help me set up Shopify" → triggers `shopify-install-auth`
- "Debug this Shopify error" → triggers `shopify-common-errors`
- "Deploy my Shopify integration" → triggers `shopify-deploy-integration`
