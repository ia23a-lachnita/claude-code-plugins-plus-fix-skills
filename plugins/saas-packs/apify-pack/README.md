# Apify Skill Pack

> Claude Code skill pack for Apify integration (18 skills)

## Installation

```bash
/plugin install apify-pack@claude-code-plugins-plus
```

## Usage

Skills trigger automatically when you discuss Apify topics. For example:

- "Help me set up Apify" → triggers `apify-install-auth`
- "Debug this Apify error" → triggers `apify-common-errors`
- "Deploy my Apify integration" → triggers `apify-deploy-integration`
