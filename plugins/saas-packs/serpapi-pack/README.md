# SerpApi Skill Pack

> Claude Code skill pack for SerpApi integration (18 skills)

## Installation

```bash
/plugin install serpapi-pack@claude-code-plugins-plus
```

## Usage

Skills trigger automatically when you discuss SerpApi topics. For example:

- "Help me set up SerpApi" → triggers `serpapi-install-auth`
- "Debug this SerpApi error" → triggers `serpapi-common-errors`
- "Deploy my SerpApi integration" → triggers `serpapi-deploy-integration`
