## Usage

Skills trigger automatically when you discuss Apple Notes topics. For example:

- "Help me set up Apple Notes" → triggers `apple-notes-install-auth`
- "Debug this Apple Notes error" → triggers `apple-notes-common-errors`
- "Deploy my Apple Notes integration" → triggers `apple-notes-deploy-integration`
