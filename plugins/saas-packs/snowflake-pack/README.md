## Usage

Skills trigger automatically when you discuss Snowflake topics. For example:

- "Help me set up Snowflake" → triggers `snowflake-install-auth`
- "Debug this Snowflake error" → triggers `snowflake-common-errors`
- "Deploy my Snowflake integration" → triggers `snowflake-deploy-integration`
