## Usage

Skills trigger automatically when you discuss Intercom topics. For example:

- "Help me set up Intercom" → triggers `intercom-install-auth`
- "Debug this Intercom error" → triggers `intercom-common-errors`
- "Deploy my Intercom integration" → triggers `intercom-deploy-integration`
