## Usage

Skills trigger automatically when you discuss ClickUp topics. For example:

- "Help me set up ClickUp" → triggers `clickup-install-auth`
- "Debug this ClickUp error" → triggers `clickup-common-errors`
- "Deploy my ClickUp integration" → triggers `clickup-deploy-integration`
