# Persona Skill Pack

> Claude Code skill pack for Persona integration (18 skills)

## Installation

```bash
/plugin install persona-pack@claude-code-plugins-plus
```

## Usage

Skills trigger automatically when you discuss Persona topics. For example:

- "Help me set up Persona" → triggers `persona-install-auth`
- "Debug this Persona error" → triggers `persona-common-errors`
- "Deploy my Persona integration" → triggers `persona-deploy-integration`
