# Cast AI Skill Pack

> Claude Code skill pack for Cast AI integration (18 skills)

## Installation

```bash
/plugin install castai-pack@claude-code-plugins-plus
```

## Usage

Skills trigger automatically when you discuss Cast AI topics. For example:

- "Help me set up Cast AI" → triggers `castai-install-auth`
- "Debug this Cast AI error" → triggers `castai-common-errors`
- "Deploy my Cast AI integration" → triggers `castai-deploy-integration`
