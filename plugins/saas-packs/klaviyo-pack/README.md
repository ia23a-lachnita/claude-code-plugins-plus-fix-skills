## Usage

Skills trigger automatically when you discuss Klaviyo topics. For example:

- "Help me set up Klaviyo" → triggers `klaviyo-install-auth`
- "Debug this Klaviyo error" → triggers `klaviyo-common-errors`
- "Deploy my Klaviyo integration" → triggers `klaviyo-deploy-integration`
