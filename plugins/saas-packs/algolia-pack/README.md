## Usage

Skills trigger automatically when you discuss Algolia topics. For example:

- "Help me set up Algolia" → triggers `algolia-install-auth`
- "Debug this Algolia error" → triggers `algolia-common-errors`
- "Deploy my Algolia integration" → triggers `algolia-deploy-integration`
