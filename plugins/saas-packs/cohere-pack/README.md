## Usage

Skills trigger automatically when you discuss Cohere topics. For example:

- "Help me set up Cohere" → triggers `cohere-install-auth`
- "Debug this Cohere error" → triggers `cohere-common-errors`
- "Deploy my Cohere integration" → triggers `cohere-deploy-integration`
