## Usage

Skills trigger automatically when you discuss Flexport topics. For example:

- "Help me set up Flexport" → triggers `flexport-install-auth`
- "Debug this Flexport error" → triggers `flexport-common-errors`
- "Deploy my Flexport integration" → triggers `flexport-deploy-integration`
