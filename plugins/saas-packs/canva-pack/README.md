## Usage

Skills trigger automatically when you discuss Canva topics. For example:

- "Help me set up Canva" → triggers `canva-install-auth`
- "Debug this Canva error" → triggers `canva-common-errors`
- "Deploy my Canva integration" → triggers `canva-deploy-integration`
