# Attio Skill Pack

> Claude Code skill pack for Attio integration (18 skills)

## Installation

```bash
/plugin install attio-pack@claude-code-plugins-plus
```

## Usage

Skills trigger automatically when you discuss Attio topics. For example:

- "Help me set up Attio" → triggers `attio-install-auth`
- "Debug this Attio error" → triggers `attio-common-errors`
- "Deploy my Attio integration" → triggers `attio-deploy-integration`
