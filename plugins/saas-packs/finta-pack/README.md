# Finta Skill Pack

> Claude Code skill pack for Finta integration (18 skills)

## Installation

```bash
/plugin install finta-pack@claude-code-plugins-plus
```

## Usage

Skills trigger automatically when you discuss Finta topics. For example:

- "Help me set up Finta" → triggers `finta-install-auth`
- "Debug this Finta error" → triggers `finta-common-errors`
- "Deploy my Finta integration" → triggers `finta-deploy-integration`
