# TechSmith Skill Pack

> Claude Code skill pack for TechSmith integration (18 skills)

## Installation

```bash
/plugin install techsmith-pack@claude-code-plugins-plus
```

## Usage

Skills trigger automatically when you discuss TechSmith topics. For example:

- "Help me set up TechSmith" → triggers `techsmith-install-auth`
- "Debug this TechSmith error" → triggers `techsmith-common-errors`
- "Deploy my TechSmith integration" → triggers `techsmith-deploy-integration`
