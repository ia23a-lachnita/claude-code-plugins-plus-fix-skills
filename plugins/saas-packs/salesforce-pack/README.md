## Usage

Skills trigger automatically when you discuss Salesforce topics. For example:

- "Help me set up Salesforce" → triggers `salesforce-install-auth`
- "Debug this Salesforce error" → triggers `salesforce-common-errors`
- "Deploy my Salesforce integration" → triggers `salesforce-deploy-integration`
