# AppFolio Skill Pack

> Claude Code skill pack for AppFolio integration (18 skills)

## Installation

```bash
/plugin install appfolio-pack@claude-code-plugins-plus
```

## Usage

Skills trigger automatically when you discuss AppFolio topics. For example:

- "Help me set up AppFolio" → triggers `appfolio-install-auth`
- "Debug this AppFolio error" → triggers `appfolio-common-errors`
- "Deploy my AppFolio integration" → triggers `appfolio-deploy-integration`
