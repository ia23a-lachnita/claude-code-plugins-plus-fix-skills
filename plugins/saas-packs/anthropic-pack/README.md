## Usage

Skills trigger automatically when you discuss Anthropic topics. For example:

- "Help me set up Anthropic" → triggers `anthropic-install-auth`
- "Debug this Anthropic error" → triggers `anthropic-common-errors`
- "Deploy my Anthropic integration" → triggers `anthropic-deploy-integration`
