## Usage

Skills trigger automatically when you discuss Miro topics. For example:

- "Help me set up Miro" → triggers `miro-install-auth`
- "Debug this Miro error" → triggers `miro-common-errors`
- "Deploy my Miro integration" → triggers `miro-deploy-integration`
