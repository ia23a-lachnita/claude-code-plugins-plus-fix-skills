## Usage

Skills trigger automatically when you discuss Ramp topics. For example:

- "Help me set up Ramp" → triggers `ramp-install-auth`
- "Debug this Ramp error" → triggers `ramp-common-errors`
- "Deploy my Ramp integration" → triggers `ramp-deploy-integration`
