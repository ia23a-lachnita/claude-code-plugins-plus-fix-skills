## Usage

Skills trigger automatically when you discuss Adobe topics. For example:

- "Help me set up Adobe" → triggers `adobe-install-auth`
- "Debug this Adobe error" → triggers `adobe-common-errors`
- "Deploy my Adobe integration" → triggers `adobe-deploy-integration`
