/plugin install notion-pack@claude-code-plugins-plus

## Usage

Skills trigger automatically when you discuss Notion topics. For example:

- "Help me set up Notion" → triggers `notion-install-auth`
- "Debug this Notion error" → triggers `notion-common-errors`
- "Deploy my Notion integration" → triggers `notion-deploy-integration`
