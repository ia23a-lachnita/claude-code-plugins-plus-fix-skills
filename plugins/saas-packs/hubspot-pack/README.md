## Usage

Skills trigger automatically when you discuss HubSpot topics. For example:

- "Help me set up HubSpot" ‚Üí triggers `hubspot-install-auth`
- "Debug this HubSpot error" ‚Üí triggers `hubspot-common-errors`
- "Deploy my HubSpot integration" ‚Üí triggers `hubspot-deploy-integration`
