```bash
/plugin install elevenlabs-pack@claude-code-plugins-plus
```

## Usage

Skills trigger automatically when you discuss ElevenLabs topics. For example:

- "Help me set up ElevenLabs" → triggers `elevenlabs-install-auth`
- "Debug this ElevenLabs error" → triggers `elevenlabs-common-errors`
- "Deploy my ElevenLabs integration" → triggers `elevenlabs-deploy-integration`
