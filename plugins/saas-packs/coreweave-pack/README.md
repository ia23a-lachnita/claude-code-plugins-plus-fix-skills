## Usage

Skills trigger automatically when you discuss CoreWeave topics. For example:

- "Help me set up CoreWeave" → triggers `coreweave-install-auth`
- "Debug this CoreWeave error" → triggers `coreweave-common-errors`
- "Deploy my CoreWeave integration" → triggers `coreweave-deploy-integration`
