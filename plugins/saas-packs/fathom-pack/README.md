# Fathom Skill Pack

> Claude Code skill pack for Fathom integration (18 skills)

## Installation

```bash
/plugin install fathom-pack@claude-code-plugins-plus
```

## Usage

Skills trigger automatically when you discuss Fathom topics. For example:

- "Help me set up Fathom" → triggers `fathom-install-auth`
- "Debug this Fathom error" → triggers `fathom-common-errors`
- "Deploy my Fathom integration" → triggers `fathom-deploy-integration`
