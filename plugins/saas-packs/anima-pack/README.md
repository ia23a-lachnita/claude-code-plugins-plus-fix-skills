# Anima Skill Pack

> Claude Code skill pack for Anima integration (18 skills)

## Installation

```bash
/plugin install anima-pack@claude-code-plugins-plus
```

## Usage

Skills trigger automatically when you discuss Anima topics. For example:

- "Help me set up Anima" → triggers `anima-install-auth`
- "Debug this Anima error" → triggers `anima-common-errors`
- "Deploy my Anima integration" → triggers `anima-deploy-integration`
