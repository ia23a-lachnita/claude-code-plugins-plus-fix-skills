## Usage

Skills trigger automatically when you discuss Palantir topics. For example:

- "Help me set up Palantir" → triggers `palantir-install-auth`
- "Debug this Palantir error" → triggers `palantir-common-errors`
- "Deploy my Palantir integration" → triggers `palantir-deploy-integration`
