# Framer Skill Pack

> Claude Code skill pack for Framer integration (18 skills)

## Installation

```bash
/plugin install framer-pack@claude-code-plugins-plus
```

## Usage

Skills trigger automatically when you discuss Framer topics. For example:

- "Help me set up Framer" → triggers `framer-install-auth`
- "Debug this Framer error" → triggers `framer-common-errors`
- "Deploy my Framer integration" → triggers `framer-deploy-integration`
