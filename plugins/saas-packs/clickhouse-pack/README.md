## Usage

Skills trigger automatically when you discuss ClickHouse topics. For example:

- "Help me set up ClickHouse" → triggers `clickhouse-install-auth`
- "Debug this ClickHouse error" → triggers `clickhouse-common-errors`
- "Deploy my ClickHouse integration" → triggers `clickhouse-deploy-integration`
