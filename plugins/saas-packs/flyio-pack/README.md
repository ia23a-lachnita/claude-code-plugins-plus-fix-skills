# Fly.io Skill Pack

> Claude Code skill pack for Fly.io integration (18 skills)

## Installation

```bash
/plugin install flyio-pack@claude-code-plugins-plus
```

## Usage

Skills trigger automatically when you discuss Fly.io topics. For example:

- "Help me set up Fly.io" → triggers `flyio-install-auth`
- "Debug this Fly.io error" → triggers `flyio-common-errors`
- "Deploy my Fly.io integration" → triggers `flyio-deploy-integration`
