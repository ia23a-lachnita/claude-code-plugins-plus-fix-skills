# Salesloft Skill Pack

> Claude Code skill pack for Salesloft integration (18 skills)

## Installation

```bash
/plugin install salesloft-pack@claude-code-plugins-plus
```

## Usage

Skills trigger automatically when you discuss Salesloft topics. For example:

- "Help me set up Salesloft" → triggers `salesloft-install-auth`
- "Debug this Salesloft error" → triggers `salesloft-common-errors`
- "Deploy my Salesloft integration" → triggers `salesloft-deploy-integration`
