## Usage

Skills trigger automatically when you discuss Procore topics. For example:

- "Help me set up Procore" → triggers `procore-install-auth`
- "Debug this Procore error" → triggers `procore-common-errors`
- "Deploy my Procore integration" → triggers `procore-deploy-integration`
