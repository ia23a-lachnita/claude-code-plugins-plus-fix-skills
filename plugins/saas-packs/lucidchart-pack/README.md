# Lucidchart Skill Pack

> Claude Code skill pack for Lucidchart integration (18 skills)

## Installation

```bash
/plugin install lucidchart-pack@claude-code-plugins-plus
```

## Usage

Skills trigger automatically when you discuss Lucidchart topics. For example:

- "Help me set up Lucidchart" → triggers `lucidchart-install-auth`
- "Debug this Lucidchart error" → triggers `lucidchart-common-errors`
- "Deploy my Lucidchart integration" → triggers `lucidchart-deploy-integration`
