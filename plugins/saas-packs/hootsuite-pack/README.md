# Hootsuite Skill Pack

> Claude Code skill pack for Hootsuite integration (18 skills)

## Installation

```bash
/plugin install hootsuite-pack@claude-code-plugins-plus
```

## Usage

Skills trigger automatically when you discuss Hootsuite topics. For example:

- "Help me set up Hootsuite" → triggers `hootsuite-install-auth`
- "Debug this Hootsuite error" → triggers `hootsuite-common-errors`
- "Deploy my Hootsuite integration" → triggers `hootsuite-deploy-integration`
